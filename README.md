# Healthie + Stripe Signup Demo

A working prototype for a custom Healthie signup flow and internal admin billing console.

## Features

- Custom athlete intake form
- Branded loading screen
- Membership tier selection
- Mock Stripe card capture flow
- Mock Healthie client creation
- Admin billing console
- Member billing table
- Charge all / charge individual actions
- Mock Vercel Cron billing route

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- LocalStorage demo data
- Mock Healthie GraphQL workflows
- Mock Stripe billing workflows

## Run Locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

Admin:

```text
/admin
```

Password:

```text
admin123
```

## Purpose

This demo was created to showcase a custom replacement for an embedded Healthie signup iframe, including membership selection, payment flow, and admin billing operations.
